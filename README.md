# Xlib
A very basic autoclicker written in C with libx11.

## Requirements
- libx11-dev
- libxtst-dev
- make

## Clone the repo
```
$ git clone https://github.com/FrancescoXD/Xlib.git
$ cd Xlib
```

## Building and running
```
$ make
$ ./xac
```

## Installation
```
$ sudo make install
```

Now you can start the autoclicker just typing `xac` everywhere. 

## Uninstallation
```
$ sudo make uninstall
```
